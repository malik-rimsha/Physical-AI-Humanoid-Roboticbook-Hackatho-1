## Glossary

- **Isaac Sim**: NVIDIA's physics-based simulation application built on Omniverse for robotics
- **Isaac ROS**: Collection of packages that bridge Isaac and ROS/ROS2 ecosystems with GPU acceleration
- **Isaac Lab**: Tools for AI training and simulation in Isaac environments
- **SLAM**: Simultaneous Localization and Mapping - technology to build maps while localizing
- **VSLAM**: Visual SLAM - SLAM using visual input from cameras
- **GPU Acceleration**: Using graphics processing units to accelerate computation
- **Synthetic Data**: Artificially generated data for training AI models
- **Domain Randomization**: Technique to improve model robustness by varying simulation conditions
- **Bipedal Motion**: Locomotion using two legs
- **Center of Mass (CoM)**: Point where the total mass of a body is concentrated
- **Zero Moment Point (ZMP)**: Point where the net moment of ground reaction forces is zero
- **Footstep Planning**: Planning where to place feet during bipedal locomotion
- **Nav2**: ROS 2's navigation framework
- **Perception Pipeline**: Processing chain from sensor data to environmental understanding
- **HDL**: Hardware Description Language (in robotics context, often refers to sensor data formats)
